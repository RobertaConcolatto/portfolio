# Portfólio - Desenvolvimento Web

Este é um projeto de portfólio pessoal desenvolvido durante a disciplina de Desenvolvimento Web.

## Sobre o projeto
Este portfólio reúne alguns dos projetos desenvolvidos ao longo do curso, utilizando HTML, CSS, Bootstrap e JavaScript.

O objetivo é praticar a criação de interfaces web, organização de layout e aplicação de conceitos de responsividade e estruturação de páginas.

## Melhorias futuras
Algumas melhorias ainda serão implementadas, como:
- Ajustes no design e identidade visual
- Melhor organização dos cards de projetos
- Otimização da responsividade em diferentes telas
- Revisão de textos e descrições dos projetos
- Possível adição de novas seções
